# BitLocker To Go FAQ

**Applies to**
-   Windows 10

## What is BitLocker To Go?

BitLocker To Go is BitLocker on removable data drives. This includes the encryption of USB flash drives, SD cards, external hard disk drives, and other drives formatted by using the NTFS, FAT16, FAT32, or exFAT file systems. 

As with BitLocker, drives that are encrypted using BitLocker To Go can be opened with a password or smart card on another computer by using **BitLocker Drive Encryption** in Control Panel.
