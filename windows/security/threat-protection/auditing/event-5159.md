# 5159(F): The Windows Filtering Platform has blocked a bind to a local port.

**Applies to**
-   Windows 10
-   Windows Server 2016


This event is logged if the Windows Filtering Platform has blocked a bind to a local port.

There is no example of this event in this document.

***Subcategory:***&nbsp;[Audit Filtering Platform Connection](audit-filtering-platform-connection.md)

***Event Schema:***

*The Windows Filtering Platform has blocked a bind to a local port.*

*Application Information:*

> *Process ID:%1*
>
> *Application Name:%2*

*Network Information:*

> *Source Address:%3*
>
> *Source Port:%4*
>
> *Protocol:%5*

*Filter Information:*

> *Filter Run-Time ID:%6*
>
> *Layer Name:%7*
>
> *Layer Run-Time ID:%8*

***Required Server Roles:*** None.

***Minimum OS Version:*** Windows Server 2008, Windows Vista.

***Event Versions:*** 0.
